Confused between inline and block elements ?

Read It Out -

Every HTML element has a display default value

There are 2 display values
1. Block
2. Inline

Basic difeerence between hese 2 displays are -

  • Block elements always starts on a new line But Inline elements does not.
  • Block element always cover the full width available BUT Inline elements only takes as much width required

Let's understand it by an example

div element is a block element and span tag is an inline element, Here I will use both tags and by using background and border property we will clearly see how they are different
